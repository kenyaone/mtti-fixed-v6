## 📦 What to Upload from the Package

### From Your Downloaded Package:

```
mtti-mis-v3.8.0-COMPLETE/
│
├── admin/          → Upload to: /wp-content/plugins/mtti-mis/admin/
├── includes/       → Upload to: /wp-content/plugins/mtti-mis/includes/
├── assets/         → Upload to: /wp-content/plugins/mtti-mis/assets/
├── public/         → Upload to: /wp-content/plugins/mtti-mis/public/
├── mtti-mis.php    → Upload to: /wp-content/plugins/mtti-mis/mtti-mis.php
│
└── verify-certificate-custom.php → Upload to: /public_html/ (ROOT!)
```

## 🎯 Two Files to Upload, Two Locations

### Location 1: Plugin Folder
**Upload To:** `/wp-content/plugins/mtti-mis/`

**What to Upload:**
- ✅ admin/ folder
- ✅ includes/ folder
- ✅ assets/ folder
- ✅ public/ folder
- ✅ mtti-mis.php file

### Location 2: Website Root
**Upload To:** `/public_html/` (same level as wp-config.php)

**What to Upload:**
- ✅ verify-certificate-custom.php

## 🔍 How to Find Your Website Root

### Via FTP:
```
Connect to FTP
→ Look for folder with wp-config.php
→ That's your root!
→ Upload verify-certificate-custom.php THERE
```

### Via cPanel File Manager:
```
Login to cPanel
→ File Manager
→ Look for public_html/ or www/ or httpdocs/
→ Inside, you'll see wp-config.php
→ Upload verify-certificate-custom.php THERE
```

### Via Hosting Control Panel:
```
Different hosts have different names:
- public_html/ (most common)
- www/
- httpdocs/
- html/
- web/

Look for the folder containing:
- wp-config.php
- wp-content/
- wp-admin/
```

## ❌ Common Mistakes to Avoid

### Mistake 1: Wrong Plugin Location
**WRONG:**
```
/wp-content/plugins/mtti-mis-v3.8.0-COMPLETE/mtti-mis.php
```

**CORRECT:**
```
/wp-content/plugins/mtti-mis/mtti-mis.php
```

### Mistake 2: Verification Page in Wrong Location
**WRONG:**
```
/wp-content/verify-certificate-custom.php
/wp-content/plugins/verify-certificate-custom.php
```

**CORRECT:**
```
/public_html/verify-certificate-custom.php
(same level as wp-config.php)
```

### Mistake 3: Uploading Package Folder Name
**WRONG:**
```
/wp-content/plugins/mtti-mis-v3.8.0-COMPLETE/
```

**CORRECT:**
```
/wp-content/plugins/mtti-mis/
(remove version number from folder name)
```

## 🚀 Quick Upload Guide

### Using FTP (FileZilla, etc.)

**Step 1: Connect to your site**

**Step 2: Navigate to plugin folder**
```
Remote site: /public_html/wp-content/plugins/
```

**Step 3: Create folder**
```
Right-click → Create directory → Name it: mtti-mis
```

**Step 4: Upload plugin files**
```
Enter mtti-mis folder
Drag and drop from local package:
- admin/ folder
- includes/ folder
- assets/ folder
- public/ folder
- mtti-mis.php file
```

**Step 5: Upload verification page**
```
Navigate to: /public_html/
Drag and drop: verify-certificate-custom.php
```

### Using cPanel File Manager

**Step 1: Open File Manager**
```
cPanel → File Manager → public_html
```

**Step 2: Go to plugins folder**
```
wp-content/ → plugins/
```

**Step 3: Create folder**
```
Click "New Folder" → Name: mtti-mis → Create
```

**Step 4: Upload plugin files**
```
Enter mtti-mis folder
Click "Upload"
Select and upload:
- All 4 folders (admin, includes, assets, public)
- mtti-mis.php file
```

**Step 5: Upload verification page**
```
Go back to public_html/
Click "Upload"
Select: verify-certificate-custom.php
Upload
```
